row

The <row> element contains a single row in a table <tgroup>.

Example

See table.

Contains

Doctype Content model
ditabase, topic, task, reference, concept, glossary, map, bookmap ( (entry) (one or more) )

Contained by

Doctype Parents
bookmap, map, ditabase, topic, task, concept, reference, glossary thead, tbody

Inheritance:

- topic/row
