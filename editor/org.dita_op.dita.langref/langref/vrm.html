vrm

The vrm empty element contains information about a single product's version, modification, and release, to which the current topic applies.

Example

See vrmlist.

Contains

Doctype Content model
ditabase, topic, task, reference, concept, glossary, map, bookmap no content

Contained by

Doctype Parents
bookmap, map, ditabase, topic, task, concept, reference, glossary vrmlist

Inheritance:

- topic/vrm

Attributes

Name Description Data Type Default Value Required?
version Indicates the released version number of the product(s) that the document describes. CDATA #IMPLIED Yes
release Contains the product release identifier. CDATA #IMPLIED No
modification Indicates when the product described in this topic was last modified. CDATA #IMPLIED No
